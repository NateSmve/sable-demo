# Sable Demo — Farm-to-Table Restaurant

A portfolio demo for SMVE Web Dev (smve.cloud). This is a **fictional** restaurant website.

## Stack
- Next.js 16 + TypeScript + Tailwind CSS v4
- Deployed to Railway

## Key Files
- `src/app/page.tsx` — Home page (hero, menu highlights, story, hours)
- `src/app/menu/page.tsx` — Full menu with sections
- `src/app/contact/page.tsx` — Reservation form
- `src/components/DemoBanner.tsx` — [DEMO] banner on every page
- `src/lib/seo.ts` — SEO metadata builder

## Rules
- All content must be clearly labeled as [DEMO]
- No fake testimonials or fake social proof
- Color scheme: cream/burgundy/gold (light theme) — uses CSS custom properties in globals.css
- Typography: Playfair Display (serif) for headings, Geist (sans) for body
